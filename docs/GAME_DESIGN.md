# Game Design: Leak Beast Arena

## Community vote result

Winner: **Leak Beast Arena**.

Interpretation:

The community wants action, mascot control, fights, Leak Beasts, bosses, and visible progress.

## Final direction

Build one main game first:

> Arena-style survival with waves, Leak Beasts, mini-bosses, progression, missions, Weekly Boss, and later clans/leaderboards.

## Core loop

1. Start run
2. Move mascot
3. Dodge attacks
4. Attack Leak Beasts
5. Survive waves
6. Face mini-boss
7. Get score / Safe Points / Boss Damage
8. Upgrade / compare / come back tomorrow

## Long-term loops

- Daily missions
- Weekly boss
- Weekly leaderboard
- Seasonal leaderboard
- Holder cosmetics
- Clan damage later

## First rule

Fun before economy.
