# MVP Scope

## v0.1 — Playable Prototype

Must have:

- mobile-sized canvas
- mascot placeholder
- movement
- attack
- dodge
- enemy waves
- score
- HP
- result screen

Must not have yet:

- real token rewards
- wallet connect
- marketplace
- clans
- paid boosts
- complex backend

## v0.2 — Arena MVP

Add:

- better mobile controls
- 3 enemy types
- mini-boss
- simple Safe Points
- daily missions
- first visual pass

## v0.3 — Life Tracker connection

Add:

- Wallet HP affects HP
- streak affects multiplier
- clean day gives shield
- Leak Score affects reward multiplier

## v1.0 — First Release

Add:

- Weekly Boss
- leaderboards
- cosmetics
- share cards
- anti-cheat limits
- Telegram Mini App integration
