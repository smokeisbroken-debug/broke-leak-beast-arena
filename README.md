# BROKE Leak Beast Arena

Playable mini-game prototype for the $BROKE ecosystem.

## Goal

Build the first real playable game layer for BROKE Life Tracker:

- control the mascot
- move, dodge, attack
- fight waves of Leak Beasts
- survive mini-bosses
- earn score / Safe Points
- later connect to Life Tracker, Weekly Boss, leaderboards, holder perks

## Current status

This repo is only the **base structure**. It is safe to fill step by step.

No backend, no token logic, no rewards, no wallet logic yet.

## Local start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Product direction

Core phrase:

> Track the leak. Fight the leak. Defeat the leak.

## First MVP scope

v0.1 should include only:

- one arena
- mascot movement
- attack
- dodge
- 2-3 Leak Beast types
- simple waves
- result screen

Do not add token rewards, clans, marketplace, or complex economy until gameplay is fun.
